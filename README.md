# 🛍️ ShopEase

ShopEase is a responsive e-commerce frontend website built using HTML, CSS, and JavaScript. This project was created to improve my frontend development skills and practice building modern, responsive web interfaces.

## 🚀 Live Demo
https://dakshitarai.github.io/Shopease/

## 💻 Technologies Used
- HTML5
- CSS3
- JavaScript

## ✨ Features
- Responsive design
- Modern and clean user interface
- Product listing section
- Navigation bar
- Shopping cart interface
- Mobile-friendly layout

## 🎯 Purpose
This project was built as a learning project to strengthen my understanding of frontend web development and responsive website design.
